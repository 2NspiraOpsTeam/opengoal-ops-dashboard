# CONTACTABLE_EMAIL_SET_V1

Generated by Adam on 2026-03-02.

## Purpose
Candidate outreach dataset for Open Goal Soccer containing contacts with email deliverability pre-checks.

## Filters Applied
- Valid email syntax
- Domain resolves via DNS
- Domain has MX records
- Deduplicated by email (nearest-distance record retained)

## Columns
- `organization_name`
- `contact_name` *(included when available in source; currently mostly blank in source data)*
- `lane`
- `distance_miles`
- `email`
- `phone`
- `website`
- `role_target`
- `source`
- `verification_flag`
- `custom_message_draft` *(starter outreach copy for Maya; editable per contact)*
- `maya_suggestion` *(Maya notes/recommendations per contact)*
- `brian_suggestion` *(Brian notes/recommendations per contact)*

## Important
This is a **candidate list**, not final approved send list. Team QA review required before production outreach.
