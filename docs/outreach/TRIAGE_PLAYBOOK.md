# Triage Playbook — Open Goal Soccer Outreach

Last Updated: 2026-03-03

## Reply Intent Tags
- **INT-1 Interested**: asks for next step, call, pricing, details
- **INT-2 Needs Info**: asks clarifying questions, wants examples
- **INT-3 Not Now**: timing issue, revisit later
- **INT-4 Not Fit / Unsubscribe**: decline, remove, wrong contact

## Fit Scoring Rubric
### High Fit
- Matches target profile and has decision influence
- Clear interest signal in current cycle
- Actionable next step requested

### Medium Fit
- Some alignment, but missing authority/timing clarity
- Interested but non-committal

### Low Fit
- Minimal alignment, wrong role, or no active need

## Routing Matrix
- **High + Interested** → Priority queue, response within 2h
- **Medium + Needs Info** → Standard queue, response same day
- **Low + Not Now** → Light nurture, scheduled revisit
- **Not Fit/Unsub** → Close thread, update suppression list

## Response Principles
- One objective per message
- One CTA per message
- Keep under 120 words when possible
- Personalize first 1–2 lines

## Logging Fields (minimum)
- Contact name
- Organization/school
- Reply date/time
- Intent tag
- Fit score
- Owner
- Next action + due date
- Outcome
