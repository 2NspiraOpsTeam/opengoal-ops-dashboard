# Social Media Advantage (SMA) — KPI Scorecard (v1)

## Reporting Cadence
- Daily pulse (light)
- Weekly scorecard (full)
- Monthly strategy review

## 1) Activity Metrics
- Candidate posts identified
- Drafts generated
- Comments posted
- Follow-up replies posted

## 2) Value/Engagement Metrics
- Reaction rate per comment
- Reply rate per comment
- Meaningful conversation rate (>=2 back-and-forth exchanges)
- Saves/shares proxy (when available)

## 3) Growth/Conversion Metrics
- OGS inquiry count
- 2Nspira inquiry count
- Book interest signals
- Click-throughs on shared links
- Affiliate link clicks
- Affiliate attributed conversions/revenue

## 4) Quality & Trust Metrics
- % comments rated high-value (internal rubric)
- Negative feedback rate
- Admin warning count
- Content rejection rate (pre-post review)

## 5) SLA Metrics
- Median discovery-to-draft time
- Median draft-to-post time
- Follow-up response compliance %
- Logging completeness %

## 6) Weekly Decision Triggers
- If negative feedback rate > 5%: reduce posting volume, tighten filters.
- If meaningful conversation rate < 15%: improve relevance and personalization.
- If affiliate CTR low but engagement high: shift to value-first/no-link pattern.
- If OGS/2Nspira inquiries rising: increase follow-up staffing windows.

## 7) Suggested Weekly Scorecard Format
- **Wins** (top 3)
- **Risks** (top 3)
- **Metrics snapshot** (table/bullets)
- **Adjustments for next week**
- **Needs decision from Jeffrey**
